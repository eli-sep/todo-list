# Todo or notTodo

A simple Todo app that renders a list of tasks from an array of objects.

## Installation instructions

After cloning this repo, open the terminal in the project's root directory and run `npm install` to install the project dependencies.

## Running the development server

To start the development server, run `npm run dev`. Then open your browser to [http://localhost:5173/](http://localhost:5173/) to view the app.
